Flare Readme

Readme

You can run the Flare API on your own computer, or in a virtual machine.

To run the server on your own computer:

Install the Node.js runtime environment.

Install the MongoDB database.

Open a terminal and change to the Flare API directory:

cd ~/Downloads/flare

Install node modules:

npm install

Start MongoDB:

mongod --dbpath data/db

Initialize the database:

node import.js model.json

Start the flare server:

node flare.js

To run the server in a virtual machine:

Install Vagrant to configure the virtualization environment.

Install the VirtualBox vitualization application.

Open a terminal and change to the Flare API directory:

cd ~/Downloads/flare

Choose a virtual box: 

vagrant init hashicorp/precise32

Start a virtual machine:

vagrant up

Start page

Open the start page in a browser:

http://localhost:1234/
